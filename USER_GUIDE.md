# OpenWebAI - 小白入门指南

> 3分钟上手，让你的本地 AI 助手免费调用 DeepSeek、元宝等大模型

---

## 这是什么？

你已经在用 Chatbox、NextChat 或其他本地 AI 客户端了对吧？但官方 API 要花钱。

**OpenWebAI** 做了一件事：**让你现有的客户端，免费使用网页版 DeepSeek、元宝等模型**。

原理很简单：你在网页上登录了账号 → 本地程序自动操控网页 → 把结果返回给你的客户端。全程无感知。

---

## 准备工作（只需一次）

### 第一步：安装 Node.js

1. 打开 [nodejs.org](https://nodejs.org)
2. 下载 **LTS 版本**（长期支持版）
3. 双击安装，一路"下一步"
4. 打开命令行（Win+R 输入 `cmd` 回车），输入：

```bash
node -v
```

看到版本号就说明装好了。

### 第二步：确认你有 Chrome 浏览器

本工具需要用 Chrome 浏览器。如果你日常用的是 Edge，也可以（Edge 底层就是 Chrome）。

---

## 快速开始（3步）

### 步骤 1：启动服务

解压下载的包后，双击 `start.bat`（Windows）或终端运行：

```bash
npm install    # 首次运行需要安装依赖（只做一次）
npm start      # 启动服务
```

看到下面这行就说明成功了：

```
✅ API 服务已启动: http://localhost:8000
```

**不要关闭这个窗口**，最小化即可。

### 步骤 2：浏览器自动打开，登录账号

程序会自动弹出一个 Chrome 窗口，打开 DeepSeek 网页。

1. **输入你的账号密码，正常登录**
2. 登录成功后，回到刚才的命令行窗口，按回车确认

> 提示：就像平时上网一样操作就行。你的账号信息只存在你自己的浏览器里，不会上传到任何地方。

### 步骤 3：配置你的 AI 客户端

打开你常用的 AI 客户端（Chatbox / NextChat / Dify 等），找到 **API 设置**，修改两项：

| 设置项 | 填写内容 |
|--------|----------|
| API 地址 | `http://localhost:8000/v1` |
| API Key | 随便填（比如 `sk-free`） |
| 模型名称 | `deepseek` |

保存设置，然后发送一条消息测试。

如果能看到回复 → **恭喜，配置完成！**

## 使用中常见情况

### 正常使用时

- 命令行窗口保持最小化运行
- 在你的 AI 客户端里正常聊天，一切跟用官方 API 没区别
- 支持多轮对话、长文本

### 遇到验证码怎么办？

偶尔网页端会弹出人机验证（Cloudflare 等），这是正常的反爬措施。

**处理方法**：
1. 程序会自动暂停并提示你
2. 切换到那个 Chrome 窗口
3. 手动完成验证（点一下"我不是机器人"）
4. 验证通过后，程序自动恢复工作

### 账号掉线了？

如果你在 Chrome 里退出了登录，或者长时间没使用导致 session 过期：

1. 程序会报错提示"登录已过期"
2. 切换到 Chrome 窗口重新登录
3. 回到命令行按回车确认

### 能同时用好几个模型吗？

MVP 版本同一时间只能用一个模型。想切换的话：

1. 关闭当前服务（Ctrl+C）
2. 编辑 `config.json`，把 `model` 改成 `"yuanbao"`（元宝）或其他
3. 重新 `npm start`
4. 重新在新打开的网页上登录对应账号

> 后续版本会支持多窗口并行，同时开 DeepSeek 和元宝。

## 常见问题

**Q: 安全吗？我的对话内容会被泄露吗？**
A: 所有数据都在你本地电脑上流转，不经过任何第三方服务器。安全性等同于你直接在网页上聊天。

**Q: 会被封号吗？**
A: 程序模拟真人打字速度（随机延迟），行为接近正常用户。但建议不要高频密集使用，正常聊天节奏没问题。

**Q: 支持 GPT-4 吗？**
A: 目前支持 DeepSeek、腾讯元宝、MiniMax。这些模型的免费版本已经很强了。后续会根据需求添加更多站点。

**Q: 我没有 AI 客户端，能直接用吗？**
A: 可以！启动服务后，用 curl 或任何 HTTP 工具直接请求：

```bash
curl http://localhost:8000/v1/chat/completions \
  -H "Content-Type: application/json" \
  -d '{"model":"deepseek","messages":[{"role":"user","content":"你好"}]}'
```
